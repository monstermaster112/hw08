# 垃圾分类助手

♻️ AI 驱动的垃圾分类智能识别系统 — 输入或语音说出物品名称，自动识别所属垃圾类别。

## 项目简介

本项目提供一个**网页应用**和一个**Python CLI 工具**，帮助用户快速查询生活垃圾的分类归属。系统内置 150+ 常见物品数据库，配合关键词规则引擎，支持文字搜索、语音输入和自然语言理解。

## 目录结构

```
垃圾分类/
├── index.html           # Web 应用（含语音识别 + 语音合成）
├── 垃圾.py              # Python CLI 工具（精确/模糊/规则三段式匹配）
├── requirements.txt     # Python 依赖说明
├── report.md            # 项目报告
├── README.md            # 本文件
└── .gitignore           # Git 忽略规则
```

## 功能特性

- **文字搜索** — 输入物品名称，自动匹配分类并显示详情
- **语音输入**（Chrome/Edge）— 点击麦克风按钮说话，支持自然语句（如「塑料瓶是什么垃圾」）
- **语音播报** — 点击喇叭图标朗读分类结果
- **分类浏览** — 按可回收物/有害垃圾/厨余垃圾/其他垃圾四类筛选
- **智能识别引擎**（Python 版）— 三段式匹配：精确 > 模糊 > 规则引擎

## 一键运行

### Web 应用（推荐）

直接双击打开 `index.html`，推荐使用 Chrome 或 Edge 浏览器。

语音识别功能需要浏览器授权麦克风权限。

### Python CLI

```bash
# 交互模式
python 垃圾.py

# 命令行模式（批量查询）
python 垃圾.py 塑料瓶 苹果核 电池 旧衣服 外卖盒
```

### 本地 HTTP 服务器（可选）

```bash
pip install flask
python -m flask run
# 浏览器访问 http://127.0.0.1:5000
```
